# Release notes: preprint public-safe preprint package

This package is a public-safe preprint candidate for the MSBP-Tg manuscript and reproducibility repository.

## Included

- Manuscript DOCX/PDF/Markdown without visible draft version labels.
- Supplementary Methods DOCX/PDF/Markdown without visible draft version labels.
- Public-safe summary tables and figures.
- Reproducibility code and tests.
- GitHub/Zenodo/ChemRxiv release workflow documentation.

## Public-safety policy

Raw third-party datasets and row-level third-party-derived SMILES/Tg tables are not redistributed. Figure 4 uses synthetic illustrative motifs rather than source-row records.

## Release workflow

Create a public GitHub repository, create a preprint-candidate release, archive it with Zenodo, then insert the real GitHub URL and Zenodo DOI into the manuscript, README, citation metadata, and release metadata before ChemRxiv posting.
- Added conservative practical design-workflow implications to the manuscript without changing code, data, or validation results.
