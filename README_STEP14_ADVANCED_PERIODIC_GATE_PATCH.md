# MSBP-Tg Step 14 Advanced Periodic Invariance Gate Patch

This patch extends the Step-13 pMSBP descriptor gate.

It adds invariance checks for:

- orientation reversal;
- cut-point / cyclic relocation;
- simple equivalent SMILES spelling;
- non-carbon simple repeat supercells;
- simple copolymer expansion.

This is not a journal-submission release. It is a descriptor-validity gate before re-analysis.
