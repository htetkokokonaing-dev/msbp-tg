# MSBP-Tg Step 18 Open-data pMSBP Statistical Re-analysis Patch

This patch computes the first open-row-level PolyMetriX pMSBP statistical re-analysis.

It includes:

- full pMSBP feature table;
- overall association;
- polymer-class and source summaries;
- within-class residual association;
- row bootstrap;
- polymer-class cluster bootstrap;
- within-class permutation test;
- leave-one-class-out sensitivity;
- conservative go/no-go verdict.

This is not a journal-submission release.
