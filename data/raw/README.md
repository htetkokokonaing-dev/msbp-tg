# Raw data directory

Raw third-party datasets are **not** stored in this public repository.

For local reproduction only, users may place independently acquired raw source files here or in a private path outside the repository. Do not commit raw source tables, row-level structural records, or row-level third-party-derived structural/property tables unless redistribution permission is explicit.

See:

- `data/external_sources.csv`
- `data/external_sources.md`
- `data/license_audit.md`
- `docs/data_provenance_protocol.md`
