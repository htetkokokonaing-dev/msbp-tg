# Processed data directory

This directory is intentionally empty in the public-safe release except for this README.

For local reproduction, after obtaining external source data under the original source terms, prepare feature tables using the documented scripts and place them here with the following expected names:

- `stage10_tsaicying_leak_excluded_novel_features.csv`
- `stage11_neurips_public_private_tg_known_novel_features.csv`
- `stage13_leeds_paek_novel_features.csv`

These files may contain row-level third-party-derived structure/property information and therefore are not redistributed in the public repository. The public release contains only aggregate validation outputs and source-role summaries.
