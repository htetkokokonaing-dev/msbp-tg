# Journal submission roadmap 

Recommended initial positioning: polymer informatics and physically interpretable boundary-coordinate analysis.

Suggested target order:

1. Digital Discovery (RSC): best fit for data-driven materials chemistry with interpretable descriptors and reproducibility.
2. Journal of Chemical Information and Modeling (ACS): good fit if the manuscript emphasizes descriptor transparency, validation, and data-driven methodology.
3. Macromolecules: possible after strengthening polymer-physics discussion and chemical structure figures.

Before submission:

- Replace repository URL placeholder after GitHub creation.
- Archive release on Zenodo and insert DOI.
- Check the selected journal's AI-assisted writing disclosure policy.
- Convert references to the target journal style.
- Update cover letter for the specific journal.
