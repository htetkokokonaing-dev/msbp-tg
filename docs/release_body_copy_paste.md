# GitHub release body preprint-candidate

Preprint-candidate release for MSBP-Tg.

Key points:

- MSBP density is transparently defined as `-NumRotatableBonds/heavy_atoms`.
- The contribution is boundary-coordinate framing and within-fiber validation, not descriptor invention.
- Main residualization protocol is within-fiber mean-centering; mean-vs-median sensitivity is reported.
- Includes manuscript, publication-draft figures, bootstrap CIs, paired descriptor comparisons, source-by-source contradiction taxonomy, and public-safe reproducibility code.
- Raw third-party datasets and row-level third-party-derived SMILES/Tg tables are not redistributed.
- Zenodo metadata is stored in `.zenodo.json`; `CITATION.cff` is included for software citation metadata.

After this release is archived by Zenodo, insert the real DOI into the manuscript, README, citation metadata, and final release metadata before creating `final DOI-bearing release`.
